# Changelog

## v0.1.0 (2026-03-14)

Initial release.

### Features

- Multi-granularity token consumption timeline (hour/day/month views)
- Model usage distribution donut chart
- Session / Subagent grouping table with expandable rows
- 5-hour window quota prediction gauge
- Real-time file monitoring with incremental JSONL parsing
- Light / Dark / System theme switching
- Multi-project support with automatic scanning
- Empty state guidance and error handling
- Windows packaging support
